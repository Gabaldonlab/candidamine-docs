CandidaMine documentation!
==================================================

CandidaMine is an integrative data warehouse for Candida Species genomes and transcriptomes. Powered by `InterMine <http://intermine.org>`_, it provides a user-friendly way to access genomic, proteomic, interaction and literature data.

This tutorial is aimed at giving users an introduction to the different parts of CandidaMine and how users can make the most of CandidaMine.

Main site: http://candidamine.org/
